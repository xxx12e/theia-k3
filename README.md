# THEIA: Learning Complete Kleene Three-Valued Logic in a Pure-Neural Modular Architecture

Code release for the paper **"THEIA: Learning Complete Kleene Three-Valued Logic in a Pure-Neural Modular Architecture"** ([arXiv:2604.11284](https://arxiv.org/abs/2604.11284)).

THEIA is a 2.75M-parameter modular neural architecture (Arithmetic / Order / Set / Logic engines with residual bridges and prototype classification) that learns all 39 rules of Kleene strong three-valued logic (K3), exhibits uncertainty–verdict asymmetric propagation verifiable by activation patching, and generalizes mod-3 sequential composition from 5-step training chains to 500-step evaluation.

All data is generated synthetically on the fly — **no datasets or checkpoints are shipped**; every artifact below is regenerated by the listed command. [CLAIM_VERIFICATION.md](CLAIM_VERIFICATION.md) contains a code-level audit of this release against the paper's Appendix B claims, including the exact optimizer configurations.

## Installation

```bash
git clone https://github.com/xxx12e/theia-k3.git
cd theia-k3

# Python 3.12 (3.12.10 used for the paper)
python -m venv .venv && .venv\Scripts\activate     # Windows
pip install --pre "torch==2.12.0.dev20260329+cu128" --index-url https://download.pytorch.org/whl/nightly/cu128
pip install -r requirements.txt
```

Reference hardware: one NVIDIA RTX 5080 (16 GB), CUDA 12.8. Any ≥12 GB CUDA GPU should work; wall-clock numbers below are RTX 5080.

**Path convention**: run everything **from the repository root**. Scripts read/write `multi_seed_results/` relative to the working directory, and shared modules live in `theia/` and `scripts/chain_pipeline/`:

```bat
set PYTHONPATH=theia;scripts\chain_pipeline        :: Windows
export PYTHONPATH=theia:scripts/chain_pipeline     #  Linux/macOS
```

## Reproducing the paper

Seeds everywhere: `{42, 123, 256, 777, 999}` (Transformer baseline extended with `{31415, 27182, 14142}`; tuned protocol uses `{42, 123, 256}`).

### Parameter counts (Appendix B) — no GPU needed
```bash
python scripts/verify_params.py
# 4-domain THEIA == 2,751,232 | chain THEIAStep == 1,508,096 | transition MLP == 4,803
```

### Table 1 / Table 2 — 4-domain K3 main result

Each architecture trains at its **development-default configuration** — THEIA: AdamW
lr 1e-3, batch 4096, cosine, 200-epoch budget; Transformer: AdamW lr 5e-4, batch 2048,
cosine, 150-epoch budget — see App. H (Correction) of the paper and
[CLAIM_VERIFICATION.md](CLAIM_VERIFICATION.md) §3.

```bash
for %s in (42 123 256 777 999) do python scripts/train_4domain/theia_5seed_v2.py --seed %s
# THEIA: ~8–9 min/seed.  Expected: 99.96% ± 0.01% overall, 5/5 seeds 12/12 Kleene.

for %s in (42 123 256 777 999) do python scripts/train_4domain/tf8l_5seed_earlystop_v2.py --seed %s
# Transformer baseline (post-LN, 3.64M) with THEIA-identical early stopping: ~51 min/seed
# when converged; seed 123 does not converge within the 150-epoch budget (as reported).

for %s in (42 123 256 777 999 31415 27182 14142) do python scripts/train_4domain/tf8l_5seed.py --seed %s
# Same model/optimizer without Kleene early stop (fixed 150-epoch budget): source of the
# Table 1 accuracy row and of the probing checkpoints. The three extension seeds
# {31415, 27182, 14142} were originally launched via scripts/train_4domain/overnight_runner.py.

for %s in (42 123 256 777 999) do python scripts/train_4domain/mlp_5seed.py --seed %s
# Flat-MLP baseline.
python scripts/train_4domain/_retrain_seed42_clean.py   # seed-42 clean retrain (Table 1 wall-clock source)
```
